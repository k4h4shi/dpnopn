# dpnopn
dpnopn - The homepage opener for npm modules that your project depend on.

## Usage
```
# install dpnopn
npm install dpnopn -g

# move to your node project root
cd your/node/project

# run dpnopn to open homepage of your project's dependency.
dpnopn
```
